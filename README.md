# ClashSub

改自于 ACL4SSR + blackmatrix7
<br/>
使用方法：随意一个订阅转换网址，将下方地址，粘贴到远程配置里面转换即可。
<br/>
```bash
https://raw.githubusercontent.com/wmsxwdsh/ClashSub/main/wmsxwdsh20230715.ini

```
